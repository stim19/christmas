# Gift Tracker

*Gift Tracker* is a holiday project built to learn and explore C++ by creating a complete, working gift-planning application. It manages people, occassions, and the gifts connected to them, with a custom SQLite engine and an ImGui interface.

## Features
- Track people and gift ideas
- Track budget with visuals
- Add occassions and link gifts to them
- Custom SQLite-backend engine
- Dear ImGui based GUI

## Why This Exists

This project was created as a personal Christmas challenge:
build one full C++ application from scratch, engine to GUI, as a way to learn architecture, memory management, and project structure.

## Requirements
- C++17 or later
- SQLite3
- GLFW
- Dear ImGui
- OpenGL3.0

## Build Instructions

```
cmake -B build
cmake --build build
```
## Running

```
./GiftTracker
```
