# Gift Tracker

*Gift Tracker* is a holiday project built using C++ by creating a complete, working gift-planning application. It manages people, occassions, and the gifts, with a custom SQLite engine and an ImGui interface.

## Features
- Track people and gift ideas
- Track budget with visuals
- Add occassions and link gifts to them
- Custom SQLite-backend engine
- Dear ImGui based GUI

## Requirements
- C++17 or later
- SQLite3
- GLFW
- Dear ImGui
- OpenGL3.0

## Build Instructions

```
cmake -B build
cmake --build build
```
## Running

```
./GiftTracker
```
